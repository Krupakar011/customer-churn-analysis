## 📌 About the Project

This project performs a complete **Exploratory Data Analysis (EDA)** on a telecom company's customer dataset to identify the key drivers behind customer churn. Using Python, Pandas, Matplotlib, and Seaborn, it uncovers actionable patterns across demographics, service usage, contract types, and payment methods.

> **Key Finding:** 26.54% of customers have churned. Month-to-month contracts, electronic check payments, and low tenure are the top churn predictors.

## 📌 Key Insights

| # | Finding | Impact |
|---|---------|--------|
| 1 | 26.54% overall churn rate | ~1 in 4 customers lost |
| 2 | New customers (1–2 months) churn most | Critical onboarding gap |
| 3 | Month-to-month contracts = highest churn | Push longer commitments |
| 4 | Senior citizens churn more | Need tailored programs |
| 5 | Electronic check users churn most | Promote auto-pay |
| 6 | No OnlineSecurity/TechSupport = higher churn | Bundle services |
